How to Publish to GitHub Pages

Step-by-step guide to putting your portfolio online for free.

Your portfolio is hosted free at yourusername.github.io using GitHub Pages.
  1. Go to github.com and create a free account if you don't have one.
  2. Create a new repository named: yourusername.github.io
  3. Upload ALL your HTML files (index.html, design.html, construction.html, permaculture.html, music.html, about.html, admin.html) to the repository.
  4. Also upload your assets/ and images/ folders.
  5. Go to Settings → Pages in your repository and set Source to main branch / root folder.
  6. Your site will be live at https://yourusername.github.io within minutes.
  7. To update the site: edit your files locally, then upload the changed files again (or use GitHub Desktop app for easier workflow).

Adding Images to Your Portfolio

Place all your photos in the images/ folder in your repository. Then reference them in project cards as images/yourphoto.jpg. For construction/permaculture, simply upload site photos to that folder.

Adding Audio Files

Create an audio/ folder and upload your .mp3 files there. Reference them in the music page as audio/yourtrack.mp3.
